⚙️ Setup Google Sheets Integration

Langkah 1: Buat Google Sheets

  1. Buka Google Sheets
  2. Buat spreadsheet baru dengan nama "TAP TAP WAR Database"
  3. Sheet akan dibuat otomatis oleh Google Apps Script

Langkah 2: Setup Google Apps Script

  1. Di Google Sheets, klik Extensions > Apps Script
  2. Hapus kode default, lalu paste kode dari file google-apps-script/Code.gs
  3. Klik Save (Ctrl+S)

Langkah 3: Deploy Web App

  1. Klik Deploy > New Deployment
  2. Pilih tipe Web App
  3. Execute as: Me
  4. Who has access: Anyone
  5. Klik Deploy dan Authorize Access
  6. Copy URL Web App yang muncul

Langkah 4: Update script.js

  1. Buka file script.js
  2. Cari bagian CONFIG.API_URL
  3. Ganti dengan URL Web App Anda:
const CONFIG = {
  API_URL: 'https://script.google.com/macros/s/YOUR_SCRIPT_ID/exec',
  // ...
};
